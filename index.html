  • TASK - Get The Weather Using An API

    Methods Used - JavaScript(internal) / JQuery / HTML / CSS (Materialize CSS internal)

    In this question, we were asked to create a single web page that is completely self-contained (i.e. does not load any external JavaScript files) and retrieve the current local weather using a free online service. There are several free weather service providers available; it was up to us to find one, study its API, figure out what endpoint to use, and write the JavaScript code to retrieve the current weather for Calgary.

    We were required to figure out how to decode the response from the API, which will most likely be a JSON-encoded object. We needed to retrieve at least the temperature and a description of the weather (e.g. partially cloudy, sunny, snowing, etc). The web page should display the required information with minimal styling.
